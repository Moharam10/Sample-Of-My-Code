Fictional University

This is one of the longest project I’ve worked on. It’s a fictional university website, a fully functional website where I integrate both back-end and front-end tools. I Build this website in parallel while I was following a web developer course on udemy.com. For me this project represents a lot because I’ve built it using Wordpress. And when I say wordpress I don’t mean customizing one of the already built-in themes and plugins. Of course I could do so, but I wanted to learn how to write and become a WordPress developer . Because I believe Wordpress currently is much more like an Web operating system and learning how to use it as a vital skill fo any developer right now. So, I decided to go for it. Of course all the credits go to Brad Schiff. And I really can’t be more grateful to him. He’s the instructor for the course Become a Wordpress developer , in this course he mentioned all what you may need to start you journey in WordPress World . And because I belive practice makes you perfect I decided to build a complete mockup for his website. The point here is that during this course and while building this website I can say I’ve learned all the basics the I needed to start with WordPress after this Course I have followed others course and now I can build fix, debug any wordpress Theme / plugin. Again thanks to Brad Schiff, I wouldn’t be able to reach this point if it’s wasn’t for Brad Scgiff.
